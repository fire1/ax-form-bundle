# AxFormBundle Documentation

Welcome to the `AxFormBundle` documentation. This bundle is designed to simplify the creation of AJAX-powered form modals and multi-step workflows in Symfony applications.

## Table of Contents

1. [Installation](installation.md) - How to get started.
2. [AxFormService](ax-form-service.md) - The fluent API for form modals.
3. [AxStepsService](ax-steps-service.md) - Managing multi-step sequences.
4. [Controller Integration](controller-integration.md) - Helper methods for your controllers.
5. [Frontend Assets](frontend-assets.md) - JavaScript, Vue components, and plugins.
6. [Twig Helpers](twig-helpers.md) - Useful Twig functions.

## Key Features

- **Fluent API**: Readable and expressive code for form handling.
- **Auto-Loading**: Automatically loads entities from the DB based on request IDs.
- **Portability**: Standalone assets that can be published as an npm package.
- **Validation Support**: Integrated server-side validation feedback inside modals.
- **multi-step Logic**: Built-in support for complex wizard-style forms.
